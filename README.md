## Chisel uses
### Chisel cutting
The Chisel can now be used to cut all of the Confluence: Otherworld Blocks into they own variants.
![Chisel recipes](https://cdn.modrinth.com/data/cached_images/a09fbed06d328944f79d6d15962fea0853318ecb.png)
### In world chiseling
The chisel can also directly cut the placed blocks into they variants. (It will not work if the blocks have a different height. By exemple: solar plate have a height of 1 but disk block have a height of 0.25, so, a solar plate can be cut into 4 disk blocks but if a solar plate are in the ground, it cannot be directly cut into a disk block.)
![Automatic Chisel Cutting](https://cdn.modrinth.com/data/cached_images/adcc3a592b83c9aae3a456ef7ff8da95161444b3.png)
